# LocalTTS

LocalTTS is an offline Text-to-Speech generator for local devices, using Facebook's MMS-TTS model. It supports English and Bangla.

## Download

[Download the latest release here](https://github.com/rifat-sarkar01/Text-to-Speech-Generator-On-Local-Device/releases/latest)

Extract the `.zip` file and run `LocalTTS.exe`. No installation is required.

## Known Issues

- **Unsigned Executable**: Windows SmartScreen might warn you on the first run. Click "More info" and then "Run anyway".
